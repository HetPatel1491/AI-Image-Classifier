# AI Image Identifier (CNN)
A full-stack web application that uses a Convolutional Neural Network (CNN) to identify images.

## Features
- **Deep Learning:** Built with PyTorch.
- **CNN Architecture:** Custom layers for pattern recognition.
- **Web Interface:** Flask-based UI for real-time uploads.
- **Accuracy:** Reached a training loss of 0.270 over 30 epochs.

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the server: `python app.py`
